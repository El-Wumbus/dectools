
Extract Supported File Types
==============================

- .tar.bz2 
- .tar.gz
- .tar
- .bz2
- .rar
- .gz
- .tbz2
- .tgz
- .zip
- .z
- .7z
- .deb
- .tar.xz
- .tar.zst
- .xz
- .shar
- .iso
> note: an iso can not be extracted, the iso will be mounted, its contents copied, and then unmounted. This reqires sudo.

To get the version number enter `extract version`
